# Stranger Things Fanpage

A fan-made website dedicated to the popular Netflix series Stranger Things.
This project showcases information about the show, including characters, storyline, and visuals inspired by the Upside Down universe.

---

## Live Website

https://strangerthingsfanpage.netlify.app/

---

## Features

* Overview of Stranger Things series
* Character / Cast showcase
* Storyline and theme presentation
* Visual banner and themed UI
* Responsive design (mobile and desktop)
* Smooth animations and layout

---

## Technologies Used

* HTML5
* CSS3
* Netlify (for deployment)

---

## Purpose of Project

This website is created as a fan project to:

* Practice frontend web development
* Improve UI/UX designing skills
* Showcase interest in the Stranger Things universe
